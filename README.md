# OpenInsight

开源社区周期洞察（Open Source Community Intelligence）系统：从多源社区信号中提炼高浓度摘要与可行动建议，并以邮件形式定时推送给团队。

## 它能做什么

- **周期简报**：按日/周生成个性化邮件简报（社区动态、风险/Breaking 预警、重要讨论与链接汇总等）。
- **多源情报**：聚合 GitHub（issue/PR/release）、论坛（如 Discourse）、官网/博客等信号，形成统一的“事件”视图。
- **深度研判**：基于opencode，以 agent 方式对关键 PR/讨论做深读，输出影响范围、潜在风险与建议动作（例如关注点、升级/回滚建议、协作机会等）。
- **可追溯性**：每封邮件附带可折叠的 `trace`（使用了哪些数据源/工具/引用链接），便于审计与复查。
- **低摩擦反馈**：用户可通过“回复一封邮件”表达偏好变化（不关注 A、多关注 B），系统据此调整后续简报侧重。

## 架构概览（规划）

系统拆分为两部分：

- `extern`：负责调度、SMTP 投递、IMAP 收信、将用户回复转为结构化反馈，并承担 PII Vault（邮箱映射与保护）。
- `opencode server`：负责数据源抓取/检索（通过 MCP）、阅读研判、生成邮件 HTML + trace、更新用户画像（以 `email_hash` 为主键）。

更完整的需求与边界说明见：`docs/prd.md`。
